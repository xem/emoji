Emoji everywhere!

If you need to use consistent color emoji in your JS13k entry or web app, you can polyfill them by loading the Twemoji-colr font made by Twitter and built by Mozilla (1.28Mb, source): 

<style>
@font-face {
  font-family: emoji;
  src: url(//xem.github.io/emoji/twemoji.ttf);
}
</style>

// Use with CSS
.emoji { font-family: emoji; }

// Use on a canvas
onload = () => {
  ctx.font = "100px emoji";
  ctx.fillText("🤟", 50, 50);
}
