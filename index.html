Color emoji webfont

If you need to use consistent emoji in your JS13k entry or web app, and worry that Emoji support is very poor across OS's and browsers, you can polyfill them by loading the Noto color emoji webfont (source): 

<style>@font-face {
  font-family: emoji;
  src: url(//xem.github.io/emoji/noto.ttf);
}</style>

// Use with CSS
.emoji { font-family: emoji; }

// Use on a canvas
onload = () => {
  ctx.font = "100px emoji";
  ctx.fillText("🤟", 50, 50);
}
